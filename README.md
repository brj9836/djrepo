## instructions

install python virtual environment and install libraries with pip

```bash
pip install -r requirements.txt
```

Migrate the database

```bash
python manage.py migrate
```

Run development server

```bash
python manage.py runserver
```

Enter the address

```bash
http://localhost:8000
```
